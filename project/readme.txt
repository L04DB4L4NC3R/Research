Hey this is Angad. Foloowing are the set of steps for running the project. Make sure you are in the same
directory as the file (applications.py)

simply run application.py by using python-3, on your terminal type-->
1. pip3 install --user -r requirements.txt
2. python3 application.py

or try this if you don't have pip3-->

1. pip install --user -r requirements.txt
2. python application.py



run the following commands on your terminal for debugger-->


1. pip install --user -r requirements.txt
2. export FLASK_APP=application.py
3. export FLASK_DEBUG=1
4. flask run
